# 使い方

## 準備
1. フォルダを新規作成します
2. 作成したフォルダのパスをクリップボードにコピーします
   - フォルダを shift + 右クリックで「パスのコピー」
   - エクスプローラのアドレスバーのパスを ctrl + c でコピー
   等
3. QuickStart.wsf を実行します
4. フォルダダイアログに「クリップボードにコピーしておいたフォルダ」が表示されるのでOKを押します
5. フォルダに必要なファイル（雛形、サンプル等）が作られます

## 作成
1. Make（ショートカット）を実行します
2. build の下に xlsx が生成されます
