// # でシート名を指定
// シート名に /,／ は使えないけど ⁄,∕,⧸,🙼 は使える
# 基本機能のサンプル

// 連続したスラッシュ記号 2 つ // からその行の末尾まではコメントです
// 文法的に無視されます
//- ※※ !!!作成中!!! まだチェックしないでください ※※

// - で項目を指定
- 項目1
- 項目2
- 項目3

// - 以外に *, + も使えます
* 項目4
+ 項目5

// 改行（複数行）
// 行末に + を書くと改行とみなし、次の行も連続した文字列として扱います
// テキストの末尾と + の間に1個以上の半角スペースが必要です
// 2行目以降の行頭のスペースは無視されます（インデントはできません）
- 複数行のサンプル +
  2行目 +
  3行目

// コメント
// 本文の後ろに [^ ○○] のように書くとコメント（Excelでマウスカーソルを合わせると表示される）になります
- コメントのサンプル（マウスカーソルを合わせてください）  [^ コメント]

// 画像コメント
// コメントの文字列を !ファイル名! のようにするとコメントに画像を表示できます
// 画像ファイルは images フォルダ以下に置いておきます
// 画像ファイルのフォーマットは excel に貼り付けれられる形式のものが使えます
// コメントと画像コメントは排他（1個のセルで同時には使えない）です
- 画像を貼るサンプル1（マウスカーソルを合わせてください）  [^ !sample1.jpg!]

// images フォルダ以下に任意のフォルダ構成を作ることもできます
// その場合 images の下からのパスで指定してください
- 画像を貼るサンプル2（マウスカーソルを合わせてください）  [^ !foo/sample2.jpg!]

// 入力欄の初期値設定
// 入力欄の順序は別途宣言しておく必要があります
// 順序宣言の記述位置について
// 1シート目よりも前に書くと全シートに対して有効になります（グローバル宣言）
// シート宣言（#の行）内に書くとそのシート内でのみ有効になります
// グローバルで estimeted_time, result の順で宣言しているので
- (15) 初期値サンプル1 +
  estimeted_time 欄に初期値 15 を設定
- (7, "< 未実装 >") 初期値サンプル2 +
  estimeted_time, result 欄にそれぞれ初期値 7, 「< 未実装 >」を設定

// 入力欄を直接指定して初期値を設定することも可能です
- (result: "- 低優先度 -") 初期値サンプル3 +
  result 欄に初期値「- 低優先度 -」を設定

// 直接指定の場合は順序宣言にない入力欄への設定も可能です
- (comment: baz) 初期値サンプル4 +
  comment 欄に初期値 baz を設定

// 行頭に【○○】と書くと、そのセルの色が変わります
// ※これは template/default.xlsx の機能です
// 条件付き初期値が設定されているので、 result 欄に - が入ります
- 【説明】説明セルのサンプル +
  「【」、「】」を含む行はチェック内容そのものではなく、説明等を書く想定


# 画像のサンプル

// パスの省略
// images フォルダ以下にフォルダを作った場合、通常であれば
// images/foo/sample2.jpg を貼る場合 [^ !foo/sample2.jpg!]
// のようにパスをすべて記述する必要がありますが、
// 変数 imagePath にフォルダのパスを指定しておくことで、
// 画像ファイルパスを記載する際にフォルダのパスを省略できます
// ただし変数 imagePath を設定できるのは 1 シートに対して 1 個だけです

// 変数 imagePath に foo というフォルダのパスを設定します
[imagePath]: foo

// imagePath が設定されていなければ [^ !foo/sample2.jpg!] と記述する必要がありますが、
// シートの imagePath に foo が設定されているので、 [^ !sample2.jpg!] のようにファイル名だけの記述で済みます
- 画像を貼るサンプル3（マウスカーソルを合わせてください）  [^ !sample2.jpg!]
